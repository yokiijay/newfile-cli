## Quick Use
```
$ npx newfile-cli src/index.js then cd
```

Will create index.js in src folder, and cd in src direcory.

## Install

`yarn global add newfile-cli`

`npm -g i newfile-cli`

## Use
```
$ newfile app/src/index.js then cd
```

## Do More
```
$ newfile somefile.js then ls -a
```
